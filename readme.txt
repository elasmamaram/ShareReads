## Project Overview  
A fully responsive book reviewing website with:
- Book catalog browsing  
- Search functionality  
- Review system  
- Admin panel  
- Random quote generator

## How to Use  
1. Unzip the project folder  
2. Open index.html in any modern browser  
3. For admin access:  
   - Click "Login" (top-right)  
   - Use credentials: admin / admin *note no spaces before or after the username and password


## Key Features Implemented  
1. *Search System* (before book listings)  
2. *Navigation Bar* (hidden in admin/login views)  
3. *Responsive Design* (works on all devices)  
4. *Interactive Elements* (reviews, quotes, etc.)

## Technical Requirements Met  
- Semantic HTML5  
- CSS3 with variables and animations  
- Vanilla JavaScript functionality  
- Proper file organization  
- Complete documentation
